Password Saver

Централизованная система хранения паролей.

- Высокая безопасность. Поддержка Advanced Encryption Standard (AES256, Rijndael);

- Экспорт списка паролей в файлы CSV;

- Хранение базы паролей частями, одной (малой) части ФП в собственных хранилищах, другой на сервере

- Генератор случайных паролей с возможностью его настройки;

Документация

- Добавление базы данных

- Добавление записей

В левом меню заполните необходимые поля вашей учетной записи: Title, Login, Password, URL. Далее нажмите [Add] - данные успешно добавлены либо вывод страницы ошибки(Error in saving). При успешно выполненной операции вы сможете увидеть изменения нажав [Open base] в левой части страницы. Теперь у вас есть новая запись в списке паролей!

- Редактирование записей

Редактирование записей происходит на отдельной странице. Сначала посмотрев записи кнопкой [Open base] выбираете чекбоксами строки для редактирования. Потом нажимаете [Edit]. В новом окне правите все записи которые хотели обновить. Нажимете [Update].

- Удаление записей

Удаление записи происходит во время просмотра базы паролей. Выбирая чекбоксами какие строки нужно удалить нажимаете кнопку [Delete]. Будьте внимательны! В данной версии продукта строки удаляются сразу после нажатия кнопки.

- Генерация паролей

Генератор паролей расположен в правой части экрана. Generate Password: используется публичный алгоритм, использующий ваш секретный ключ(Secret) и адрес сайта(URL). Также можно указать длину пароля(Max_length). Далее нажимаете [Generate] - пароль будет отображен ниже, можете его скопировать.
